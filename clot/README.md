# Clot Package Root

`clot/` is the canonical package root.

Recommended imports:
- `import clot.science.utils;`
- `import clot.core.helpers;`
- `import clot.io.files;`
- `import clot.ml.machine_learning;`

Layout:
- `clot/science/<module>/<module>.clot`
- `clot/core/<module>/<module>.clot`
- `clot/io/<module>/<module>.clot`
- `clot/ml/<module>/<module>.clot`
