## Sidebar organization

The navigation sidebar on the left is organized to mirror a typical developer's journey:

* **Getting Started:** The essentials to install the language and run your first scripts.
* **Core Modules:** Deep dives into language features, syntax, and standard libraries.
* **Advanced & Contributing:** Lower-level details regarding the LLVM backend and static analyzer for contributors.

<Callout type="info">
  **Pro tip:** You can collapse sections in the sidebar to keep your view clean, or use the search bar (`Ctrl + K`) to bypass navigation entirely.
</Callout>
